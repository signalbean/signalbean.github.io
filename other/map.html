Paper Map Navigation System

This project is a hybrid paper-to-digital navigation system. The physical maps shown below are designed for real-world use, while each QR code provides instant digital access for quick reference, sharing, and mobile viewing. The goal is to combine the reliability of printed maps with the convenience of modern web access.
