Aki Bot – FAQ

Common questions about using Aki, its commands, and features.

What is Aki Bot?

Aki is a Discord bot designed for fetching images from Danbooru. It allows users to search for images using tag filters, manage content ratings, and create custom commands for frequently used tags.

What are the main commands available with Aki Bot?

Aki Bot has several commands:

  • /fetch: Used to get random Danbooru images with options for search tags and content ratings, or to fetch an image by its post ID.
  • /add: Allows users to create custom tag shortcuts (e.g., /neko for cat_girl).
  • /list: Displays all custom tag commands created in a server.
  • /remove: Deletes existing custom tag commands.
  • /help: Provides general information about using the Aki bot, its commands, usage examples, and content ratings.

There's also context menu commands, Which also have useful stuff.

How do content ratings work with Aki Bot?

Aki Bot uses four content ratings: Safe (G), Sensitive (S), Questionable (Q), and Explicit (E). In SFW (Safe For Work) channels, only Safe content is available. In NSFW (Not Safe For Work) channels, all ratings (Sensitive, Questionable, Explicit) are available, with Explicit being the default rating. The bot also applies automatic blacklist filtering and quality control.

Can I create my own custom commands?

Yes, you can create custom tag commands using the /add command. You need to provide a name for your new command (e.g., "neko") and the Danbooru tag it should search for (e.g., "cat_girl"). You can also add an optional description. To manage custom commands, users need the "Manage Messages" permission.

What are the limitations when using Aki Bot?

When using the /fetch command, there is a maximum limit of one search tag due to API limitations. The bot automatically filters out inappropriate content based on a blacklist of tags, and it only supports PNG/JPG image formats. Additionally, custom tag commands cannot use names reserved for built-in commands like fetch or help.
